Estrutura Básica

<html> </html>

Para dar a estrutura é necessário incluir a tag html assim pode ser
incluídos outras tags pois entendido que dentro destas tags ali haverão
outras tags html, ou seja, define a raiz do arquivo HTML.

<head> </head>

Certas informações precisam ser adicionadas antes do lado client ser totalmente carregado, as informações de linguagem padrão por exemplo, esses metadados são informações do documento.

<body> </body>

A estrutura só pode ser feita com um molde e este é a tag "body" realizando o papel de definir o corpo das estruturas, assim mostrando-as para os usuários.

Tags

Como em todo início de algo é preciso mostrar um caminho, foi mostrado algumas tags para dar exemplo a respeito do funcionamento das mesmas no HTML.

<input type = "text"/>

Muito utilizado em formulários para receber todo tipo de informação, definindo o modo de entrada de dados com seus atributos e mais algumas informações extras que auxiliam na configuração dos atributos

<img src="" width="150" alt="">

Exemplo:

Para se inserir imagens usamos a tag "img", além de possibilitar a inserção da imagem podemos editar seu tamanho com atributos

Atributos

São formas de aidionar regras extras a tags já criadas como por exemplo mudar o tamanho da imagem na tag "img" com o atributo width

<img src="" width="" alt="">
Exemplo:

    Imagem original:

    • Imagem Alterada com width:

Outra tag que usamos atributos é a tag de ancoragem <a> onde colocamos um atributo para o link ser aberto em outra página com o atributo "target"

<a href="link desejado" target="blank">texto clicavel</a>

Tags de texto

Aqui vou exibir as tags de texto como esta que estou escrevendo, a tag de parágrafo de texto <p>, exemplo abaixo:

<p>texto como o acima</p>

O próprio negrito do exemplo é outra tag de texto <strong> </strong>, e abaixo vou mostrar a tag de itálico <i>,junto a um exemplo.

<i>texto em itálico</i>

Também utilizei a tag de sublinhar o underline <u>, mas mostrarei duas tags parecidas a <sup> e <sub>.

<sup>texto em sup</sup> <sup>texto em sub</sup>

Como pode der visto a tags podem tornar textos em superescrito ou sobrescrito, e usei a tag <mark> para ressaltar o texto dentro das tags e abaixou vou mostrar outro parecido que destaca o texto de uma maneira diferente.

Este é a citação chamada <blockquote></blockquote>
